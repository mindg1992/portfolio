📊 2. 포트폴리오 입력 방법
① 종목 추가
연도 헤더의 + 종목 추가 버튼 클릭 → 종목명, 자산 유형, 목표비중(%), 기대수익률 입력 후 추가
• 종목을 클릭하면 수정할 수 있어요  |  ⋮⋮ 핸들을 드래그하면 순서 변경 가능
② 자산액 입력
각 월 칸을 클릭하면 계산기 팝업이 열려요. 숫자를 입력하거나 수식(예: 3000+500)도 계산 가능해요.
• 대출은 자동으로 음수(-) 처리돼요
③ 적립금 입력 ← 이 부분이 핵심!
📌 올바른 사용 순서
1 해당 월의 종목별 현재 자산액을 먼저 입력
예) 2월 열에 각 종목의 현재 평가금액 입력
2 그 달 헤더의 "적립금" 란에 이번 달 추가 투자 예정 금액 입력
예) 이번 달 월급에서 투자할 금액 400만원 입력
✓ 각 종목 아래에 목표비중에 맞게 얼마씩 사야 하는지 자동 계산!
예) A종목에 150만원, B종목에 250만원 매수 안내
💡 적립금은 "이번 달에 새로 투자하는 돈"이에요. 이미 보유한 자산액과 분리해서 입력해야 정확한 분배 안내가 나와요.

⌨️ 4. 알아두면 좋은 조작법
🖱️ 기본 조작
• 자산 셀 클릭 → 계산기 팝업 오픈
• 종목명 클릭 → 종목 정보 수정
• ⋮⋮ 드래그 → 종목 순서 변경
• 목표비중 칸 직접 입력 가능
• 목표비중 합계가 100%가 되어야 정상
⌨️ 계산기 단축키
• Enter → 입력 확정
• Esc → 취소
• Backspace → 한 글자 삭제
• + - * / 수식 계산 지원
• 예) 3000+500 입력 후 Enter
📅 연도 & 월 관리
• + 연도 추가 → 새 연도 생성
• + 다음 달 추가 → 해당 연도에 월 추가
• 월 삭제는 가장 최근 달부터만 가능
• 첫 연도 클릭(✎) → 시작 연도 변경
📈 수익 계산 방식
• 순수익 = 이번달 합계 - 지난달 합계 - 지난달 적립금
• 적립금을 빼야 진짜 투자 수익률이 나와요
• 대출은 자산에서 자동 제외됩니다
• 비중 제외 종목은 합계·비중에서 빠져요
